# McJohnson

**Privacy-first mobile apps and a curated space for tech recommendations.**

[mcjohnson.website](https://mcjohnson.website) — home of Pefi, Yearbook, and honest tech picks.

---

## About

McJohnson is a one-person indie studio based in Poland, EU. The apps are built around one principle: your data stays on your device, always. No cloud sync, no accounts, no tracking inside the apps.

This repository contains the source code for the public website.
